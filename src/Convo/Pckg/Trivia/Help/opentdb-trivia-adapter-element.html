OpenTDB Trivia Adapter Element

Transforms a given OpenTDB quiz into a format more suitable for Convoworks usage.

When to use?

If you want to use an OpenTDB quiz with Convoworks Trivia elements (e.g. trivia round block), and you don't want to have to manually transform questions into the proper format.

How to use?

  • First, pick a session type and name under which the transformed questions will be stored.
  • Choose an amount of questions you want to fetch.
  • Pick a category code. Here's a (potentially non-exhaustive) list of categories and their codes:
    Category Code
    Film 11
    Video games 15
    Animals 27
    Music 12
    History 23
    Geography 22
    Sports 21
    Television 14
  • The resulting questions have the following properties
    • text — The actual text of the question.
    • answers — An array of answer objects, which look like this:
      • text — The text of the answer
      • letter — The letter associated with the answer
      • is_correct — Boolean; whether the answer is correct or not.
    • As a quality-of-life feature, each question has a correct_answer object that is set to the correct answer. Refer to the layout of answer objects above.
  • In the OK flow, place the elements you wish to run if the quiz has been successfuly loaded.
  • In the Not OK flow, place the elements you wish to run if an error occurs and the quiz could not be loaded.
